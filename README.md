**Variable Descriptions in Final CSV**

Not_graduated — Percentage of adults aged 25 and older who did not complete high school.

HDI — Human Development Index assigned to each state.

Health_Index / Education_Index / Income_Index — Component indices used in calculating the HDI, representing health, education, and income outcomes for each state.

Homelessness_Ratio — Number of people experiencing homelessness per 10,000 residents.

Unsheltered_Homelessness — Percentage of the homeless population that is unsheltered.

Avg_PM25 — Average concentration of fine particulate matter (PM2.5), which refers to airborne particles 2.5 micrometers or smaller. Values were averaged from a random sample of 20 air-quality sensors in each state.
